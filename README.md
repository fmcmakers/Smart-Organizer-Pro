**Smart Organizer Pro** is a powerful, multithreaded desktop application designed to help digital content creators and professionals automatically organize their files, analyze disk space, and find duplicates safely.

## ✨ Features
* **🚀 Smart Sorting:** Automatically categorizes files into structured folders (Images, Videos, Documents, etc.) based on modification dates.
* **🕵️ Duplicate Finder:** Uses MD5 hashing to detect and safely move duplicate files to a dedicated folder.
* **📊 Live Dashboard:** Real-time disk space analysis without freezing the UI (Multithreaded).
* **🛡️ System Shield:** Safely ignores system directories (`Windows`, `Program Files`, `$RECYCLE.BIN`, etc.) to prevent accidental OS corruption.
* **⏪ Emergency Undo:** One-click rollback to restore files to their original locations if a mistake happens.
